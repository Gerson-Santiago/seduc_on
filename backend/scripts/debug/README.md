# ⚠️ Scripts de Debug

**ATENÇÃO: ESTA PASTA NÃO DEVE IR PARA PRODUÇÃO!**

Os scripts contidos nesta pasta são ferramentas auxiliares para desenvolvimento e debug local. Eles podem conter lógica que altera dados diretamente no banco ou expõe informações sensíveis se usados incorretamente.

## Regras
1. **Não commitar scripts**: Apenas este README deve ser versionado.
2. **Não usar em produção**: Estes scripts não passam pelos mesmos controles de segurança da aplicação principal.
3. **Cuidado com dados**: Se criar scripts que manipulam dados reais, certifique-se de sanitizá-los.
